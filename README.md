# Forest Creek Lodge - Booking System
## Frontend HTML & CSS Project

A professional, responsive website for a luxury lodge booking system based in Mutare, Zimbabwe.

---

## 📁 Project Structure

```
BOOKING LODGE/
├── index.html          # Home/Landing page with featured rooms
├── rooms.html          # All accommodations with detailed descriptions
├── booking.html        # Booking form with multiple fields
├── about.html          # About us, mission, team, and values
├── contact.html        # Contact info, contact form, gallery & FAQ
├── css/
│   └── styles.css      # Main stylesheet (green & earth tone theme)
├── images/             # Folder for lodge photos (empty - add your images)
├── js/                 # Folder for future JavaScript functionality
└── README.md           # This file
```

## 🚀 How to Use

1. **Open in Browser**: Simply open `index.html` in any web browser
   - Chrome, Firefox, Safari, Edge all supported

2. **Navigation**: Use the top navigation menu to jump between pages

3. **Responsive Design**: Resize your browser to test mobile responsiveness

4. **Forms**: Contact and booking forms are set up but will need backend processing

## 🔧 CSS Variables Used

All colors are defined as CSS variables in `:root` for easy customization:

```css
--primary-green: #2d5016;
--sage-green: #9caf88;
--earth-brown: #8b7355;
--light-cream: #f5f5f0;
--dark-text: #3e3e3e;
--accent-green: #5a7c3e;
--border-color: #ddd;
```
